よくある質問 (FAQ)

Q: サーバーが見つかりません

A: サーバーとiPhoneが同じWi-Fiネットワークに接続されているか確認してください。また、サーバーが正常に起動しているか、ファイアウォールでポートがブロックされていないか確認してください。

Q: 「証明書が無効です」と表示されます

A: サーバーを再起動して新しいフィンガープリントを確認し、iOSアプリでサーバーを削除して再登録してください。

Q: CLIが実行されません

A: サーバーマシンでCLI（claude/codex/gemini）が正しくインストールされ、認証設定が完了しているか確認してください。

Frequently Asked Questions (FAQ)

Q: Server not found

A: Verify that the server and iPhone are connected to the same Wi-Fi network. Also check that the server is running properly and the port is not blocked by a firewall.

Q: "Invalid certificate" error

A: Restart the server to get a new fingerprint, then remove and re-add the server in the iOS app.

Q: CLI not executing

A: Verify that the CLI (claude/codex/gemini) is correctly installed on the server machine and authentication is configured.
